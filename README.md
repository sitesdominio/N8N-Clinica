# N8N-Clinica
Este projeto automatiza processos de atendimento em clínicas, integrando comunicação via WhatsApp com Inteligência Artificial para otimizar o fluxo de agendamentos e triagem.

Tecnologias Utilizadas
* n8n
* Groq / Gemini API
* Uazapi Integração com WhatsApp.
* Contabo VPS

Como usar
1. Importe o arquivo `.json` presente neste repositório para o seu n8n.
2. Configure as credenciais das APIs de IA e WhatsApp.
3. Ative o workflow.
